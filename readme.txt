A dynamic Wordpress sidebar widget to display the Hacker symbol

== Description ==

gLider sidebar displays the glider logo in your wordpress sidebar

== Installation ==

1. Upload `glider.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Appearance -> Widgets and drag the widget into your sidebar
4. Customise the widget's settings, save, and enjoy! :)
